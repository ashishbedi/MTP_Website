Abstract

The project is focussed towards presenting a novel approach for object identification, annotation and tracking in a drone video. The idea is to make the system first identify the object(s) of interest in the video, either automatically or by prompting the user for well defined boundary(ies). It then tries annotating the object based upon models learned through supervised training. It further tracks the given object(s). It involes image segmentation with deep learning, use of CNN and RCNN and techniques like background removal to focus on the object. STRUCK paper has been implemented for object tracking, which makes use of kernel methods and Support Vectors to track the object of interest.

Motivation

  1. To see how a model trained on datasets of various images of different objects works in identifying an object.
  2. Many use cases, ranging from tracking a thief to tracking a bird.
  3. Tracking objects in live feed.

Objective

Given a video, identify the object, annotate and label it, and track the object.

Keywords:
